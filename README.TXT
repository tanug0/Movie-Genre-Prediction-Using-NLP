# Movie Genre Predictor

## Description
Movie Genre Predictor is a Machine Learning and NLP project that predicts the genre of a movie using its description.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- Streamlit

## Features
- Text Preprocessing
- Genre Prediction
- Model Evaluation
- Streamlit Interface

## Run

python movie_genre.py

streamlit run app.py
